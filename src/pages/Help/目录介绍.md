## 目录介绍



```
|- dist                 // 打包后的文件
|- plugins              // 自定义Vue插件
|- src                  // 开发主目录
|- |- components        // 平台组件，非表单组件
|- |- func              // 表单组件
|- |- less              // less文件
|- |- pages             // 页面文件
|- |- store             // Vuex配置文件
|- |- utils             // 工具包
|- |- App.vue           // 入口Vue文件
|- |- index.js          // 入口index.js文件
|- |- index-dev.html    // 开发环境的html
|- |- index-prod.html   // 生产环境的html
|- └ router.js          // vue-router配置文件
|- LICENSE.md           // LICENSE许可证
|- README.md            // 项目的说明文档，markdown格式
|- package.json         // 项目配置文件
|- package-lock.json    // 项目配置文件
|- fish.config.js       // 自定义打包工具配置文件
|- favicon.ico          // 网站ico
|- .gitignore           // git提交忽略文件
└.editorconfig          // 代码自动格式化规范
```
