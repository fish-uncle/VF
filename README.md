## 介绍

VF，即Vue Form，一个为Vue而设计的可视化表单设计器。

基于 vue 和 [view-design](https://www.iviewui.com/) 实现开发的，内置了10多余个基础组件，可扩展性的自定义组件，可以让表单开发简单而高效。

下面就会把本项目的使用方式本着简单，易用的原则，一一讲解给大家。
